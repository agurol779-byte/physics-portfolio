# Ali Kemal Gurol | Popular Science & Physics Portfolio

This is a dynamic web portal showcasing my latest popular science articles and physics research.

## 🚀 Features
- **Dynamic Content:** Articles are fetched in real-time from a cloud database (JSONBin).
- **Physics Visuals:** Interactive background with physics formulas and particle effects.
- **Admin Panel:** A custom-built management unit to add or remove articles directly from the UI.
- **Responsive Design:** Dark/Light mode support with a modern scientific aesthetic.

## 🛠️ Tech Stack
- HTML5, CSS3, JavaScript (ES6+)
- JSONBin API for data management
- Vercel for cloud deployment
